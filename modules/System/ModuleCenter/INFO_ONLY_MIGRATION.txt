Migration: Info.php + InstallController only (no legacy manifests)

Changes:
- Modules are discovered ONLY if Config/Info.php exists.
- Module install/uninstall/update are executed via Controllers/InstallController.
- Menu arrays in files are deprecated; MenuService no longer imports menu items from manifests.
- Each module that wants a system admin menu entry must implement Models/InstallModel::addMenu()/delMenu() and call it from InstallController.

What you must do for each module (pages/blocks):
1) Create Config/Info.php with:
   public string $name, $title, $version; public int $weight;
2) Create Controllers/InstallController.php with methods:
   install():bool, uninstall():bool, update(string $from,string $to):bool
3) Create Models/InstallModel.php with addMenu()/delMenu() if menu entry needed.

Legacy Config/Module.php is ignored.

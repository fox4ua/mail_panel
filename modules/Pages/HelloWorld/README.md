# HelloWorld (Pages module) v2

Исправление под вашу текущую реализацию меню:
- В `Config/Module.php` в пункте меню `route` теперь **URI**, а не имя роут-алиаса.
  То есть будет ссылка на `/hello`, даже если меню строит ссылку через `site_url($route)`.

Роут всё равно имеет алиас `helloworld.index` (на будущее, если вы захотите генерировать URL через `route_to()`).

Проверьте:
- /hello
